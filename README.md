
# 📊 Banking Customer Data - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a banking customer dataset using **Python**. The goal is to clean, analyze, and visualize customer data to uncover meaningful insights into customer demographics, financial behavior, and banking patterns. The project demonstrates practical data analysis skills using Python libraries and was developed in **Visual Studio Code (VS Code)**.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing.
* Analyze customer demographics and financial information.
* Handle missing values and inspect data quality.
* Explore relationships between banking attributes.
* Create meaningful visualizations.
* Generate business insights from customer data.

---

## 🛠️ Tech Stack

* **Python**
* **Visual Studio Code (VS Code)**
* **Pandas**
* **NumPy**
* **Matplotlib**
* **Seaborn**

---

## 📂 Dataset Features

The dataset includes customer banking information such as:

* Customer Age
* Occupation
* Nationality
* Estimated Income
* Risk Weighting
* Loyalty Classification
* Properties Owned
* Joined Bank Date
* Superannuation Savings
* Credit Card Balance
* Bank Loans
* Bank Deposits
* Checking Accounts
* Savings Accounts
* Foreign Currency Account

## 📋 Project Workflow

### 1. Data Collection

* Load the dataset using Pandas.
* Inspect the dataset structure and dimensions.

### 2. Data Cleaning

* Check for missing values.
* Verify data types.
* Convert date columns where required.
* Remove duplicates (if any).

### 3. Feature Engineering

* Create income categories (Low, Mid, High).
* Prepare data for analysis.

### 4. Exploratory Data Analysis

* Summary statistics
* Distribution analysis
* Correlation analysis
* Categorical variable analysis
* Financial behavior analysis

### 5. Data Visualization

* Histograms
* Count Plots
* Box Plots
* Scatter Plots
* Correlation Heatmap

### 6. Business Insights

* Identify customer trends.
* Analyze banking product usage.
* Understand relationships between financial variables.

---

## 📈 Key Insights

* Higher-income customers generally maintain larger bank deposits.
* Savings and checking account balances exhibit positive relationships.
* Customer segmentation based on income helps understand banking behavior.
* Correlation analysis identifies important financial relationships useful for business decision-making.
